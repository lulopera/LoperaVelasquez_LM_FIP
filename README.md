# Alpaca Amigurumi Store

## Description
   Alpaca Amigurumi Store is a store of handmade toys created with yarn. The technique used in creating this toys is called "Amigurumi", which is a Japanese art of knitting and crocheting small figures. 
   The toys are made with 100% cotton, are non-allergenic and  are washables in the washing machine.
   You can buy your amigurumies in the store shop paying with credit card or PayPal.
